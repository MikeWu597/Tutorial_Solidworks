# 7. 阵列与圆角

阵列和圆角对于添加细节和制造考虑因素至关重要。

## 线性阵列
1. 选择要阵列的特征

我们需要将前文的旋转切除复制到另一个侧面：

![](007-1.png)

2. 单击"线性阵列"
3. 设置：
   - 方向（边线或轴）
   - 实例数量
   - 实例之间的间距

![](007-2.png)

4. 单击确定以创建阵列

![](007-3.png)

## 圆周阵列
1. 选择要阵列的特征
2. 单击"圆周阵列"
3. 设置：
   - 旋转轴
   - 实例数量
   - 分布角度
4. 单击确定以创建阵列

## 圆角
圆角在面之间创建圆角过渡。

1. 单击"圆角"工具

![](007-4.png)

2. 设置圆角半径
3. 选择要倒圆角的边线

![](007-5.png)

4. 单击确定应用圆角

![](007-6.png)

## 圆角类型
- **等半径**：所有选定边线的相同半径
- **变半径**：沿边线的不同半径
- **面圆角**：两个面之间的圆角
- **完整圆角**：在面之间创建滚动球圆角
