# 6. 旋转与旋转切除

旋转通过围绕轴旋转 2D 草图来创建 3D 特征。

## 旋转凸台/基体（切除）
1. 创建包含以下内容的草图：
   - 要旋转的轮廓
   - 中心线（旋转轴）
   
![](006-1.png)
   
创建中心线时，可以选择“作为构造线”、“无限长度”，这将创建一个构造线，该线将用于旋转轴。

![](006-2.png)

2. 单击"旋转切除"特征

![](006-3.png)

![](006-4.png)

3. 设置参数：
   - 旋转角度（0° 至 360°）
   - 旋转方向
   - 旋转轴（通常是中心线）
4. 单击确定以创建特征
5. 设置类似于旋转凸台/基体的参数
6. 单击确定以创建切除

![](006-5.png)

## 重要说明
- 草图轮廓必须位于中心线的一侧
- 自相交的轮廓会导致错误
