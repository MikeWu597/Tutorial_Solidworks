# 4. 圆形和弧线

圆形和弧线对于创建曲线几何体至关重要。

## 圆形类型
1. **圆**：圆心点和半径
2. **周边圆**：圆周上的三点
3. **相切圆**：与现有实体相切的圆

## 创建圆
1. 选择"圆"工具
2. 单击放置圆心点
3. 移动光标以定义半径并单击
4. 添加尺寸以控制大小

![](004-2.png)

## 弧线类型
1. **圆心点弧**：通过圆心、起点和终点定义
2. **相切弧**：与现有实体相切的弧
3. **3 点弧**：通过弧上的三点定义

## 使用弧线
1. 选择适当的弧线工具
2. 定义所需的点并作出圆弧

![](004-1.png)

3. 添加尺寸以确定半径和角度范围

![](004-3.png)

4. 应用关系以控制位置和相切性
