# 5. 拉伸与拉伸切除

拉伸是将 2D 草图转换为 3D 实体的主要方法。为了便于演示，我们首先作出如图的草图：

![](005-1.png)

接下来，我们用等距实体功能作出等距的框线，这样，草图将形成环状：

![](005-2.png)

![](005-3.png)

![](005-4.png)

## 拉伸凸台/基体
1. 创建并退出草图
2. 单击"拉伸凸台/基体"特征

![](005-5.png)

3. 设置拉伸深度：
   - 盲孔：指定距离
   - 贯穿所有：拉伸贯穿所有几何体
   - 到顶点/面：拉伸到特定点

![](005-6.png)

![](005-7.png)

此处我们填写50mm以便后续演示。

![](005-8.png)

4. 设置方向（垂直于草图或特定方向）
5. 单击确定以创建特征

## 拉伸切除
1. 在现有面上创建草图

![](005-9.png)

点击侧面位置。

2. 绘制要切除的轮廓

在侧面上绘制孔洞的位置：

![](005-10.png)

3. 单击"拉伸切除"特征
4. 设置类似于拉伸凸台/基体的参数：
   - 切除深度
   - 方向
   - 贯穿所有选项

![](005-11.png)

5. 单击确定以创建切除

![](005-12.png)

## 关键选项
- **方向 1/方向 2**：在一个或两个方向上拉伸
- **拔模角度**：锥形拉伸

![](005-13.png)
