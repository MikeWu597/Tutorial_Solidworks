# 3. 直线与矩形

让我们从基本的草图实体开始：直线和矩形。

## 创建直线
1. 单击草图工具栏中的"直线"工具
![](003-1.png)

2. 单击以定义起点
3. 移动光标并单击以定义终点
4. 继续单击以创建连续的直线
5. 双击或按 Esc 键完成

![](003-2.png)

## 创建矩形
SolidWorks 提供了几种矩形工具：
1. **角点矩形**：通过两个对角点定义

![](003-3.png)

2. **中心矩形**：通过中心点和角点定义
3. **3 点角点矩形**：通过三个角点定义
4. **3 点中心矩形**：通过中心和两边上的两点定义

## 添加尺寸
1. 选择"智能尺寸"工具

![](003-4.png)

2. 单击要标注尺寸的线条或矩形边
3. 通过移动光标放置尺寸
4. 输入所需值或接受当前测量值

![](003-5.png)

要确定一个图形在基准面中的位置，可以与辅助线交点作距离限制。

![](003-6.png)

这样，草图图形就显示为黑色，代表所有自由度都被完全消除，处于完全定义状态。此时草图完全严谨。

![](003-7.png)

## 添加关系
关系是自动的几何约束，例如：
- **水平/垂直**：强制线条完全水平或垂直
- **垂直/平行**：建立角度关系
- **重合**：使点或实体接触
- **相等**：使实体大小相同

添加关系的方法：
1. 选择要关联的实体
2. 查看属性管理器中的建议
3. 或从"添加关系"工具中手动选择关系
