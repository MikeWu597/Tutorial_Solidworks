# 9. 材质、质量属性与干涉检查

分配材料和检查物理属性对于设计验证至关重要。

## 分配材料
1. 右键单击特征管理器设计树中的"材料"

![](009-1.png)

2. 选择"编辑材料"
3. 从材料库中选择：
   - 金属（钢、铝等）
   - 塑料（ABS、聚碳酸酯等）
   - 自定义材料
   
![](009-2.png)

此处，我们选择ABS。

4. 单击应用以分配材料

## 检查质量属性
1. 单击"评估" > "质量属性"
2. 查看信息包括：
   - 体积
   - 质量
   - 表面积
   - 重心
   - 惯性矩
3. 将这些数据用于工程计算

![](009-3.png)

## 干涉检查
在装配体中，检查组件干涉：
1. 打开装配体
2. 单击"评估" > "干涉检查"

![](009-4.png)

3. 设置选项：
   - 将重合面视为干涉
   - 包括多实体零件
4. 运行检测以查找重叠组件

![](009-5.png)

5. 通过调整组件位置或尺寸解决干涉
